## JSX Advanced

Now let's explore more advanced JSX patterns including conditional rendering, lists, and styling.

### Rendering Lists

First, let's look at how to render a list of items. We'll start with an array of todo items:

```jsx
export function App() {
  const todos = [
    { id: 1, text: "Learn Kaioken", done: false },
    { id: 2, text: "Build an app", done: true },
    { id: 3, text: "Share with friends", done: false }
  ]
  
  return (
    <ul>
      {todos.map(todo => (
        <li key={todo.id}>{todo.text}</li>
      ))}
    </ul>
  )
}
```

Notice the `key` prop - this is important for Kaioken to efficiently update the DOM when the list changes.

### Dynamic Styling

JSX allows us to apply styles conditionally:

```jsx
<li style={{ 
  textDecoration: todo.done ? 'line-through' : 'none',
  color: todo.done ? '#666' : '#000'
}}>
  {todo.text}
</li>
```
