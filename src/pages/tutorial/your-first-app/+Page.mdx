## Let's create your first app!

Time to do cool things!

First, let's edit ``App.tsx`` and declare a ``name`` variable:

```jsx
export default function App() {
  const name = "Kaioken" // [!code ++]
  return <h1>Hello world!</h1>
}
```

Then, we can refer to ``name`` in the markup:

```jsx
export default function App() {
  const name = "Kaioken"
  return <h1>Hello |~~{name}~~|!</h1>
}
```

Inside the curly braces, we can put any JavaScript we want. 
Try changing ``name`` to ``name.toUpperCase()`` for a shoutier greeting.

```jsx
export default function App() {
  const name = "Kaioken"
  return <h1>Hello {name|++.toUpperCase()++|}!</h1>
}
```
