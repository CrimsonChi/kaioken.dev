# useEffect

Runs a function after the component is rendered, or 
when a value provided in the optional 
<a href="/docs/hooks/dependency-arrays">dependency array</a>
has changed.

### Example
<div className="not-prose">
  <CodeBlock 
    lang="jsx" 
    code={`\
import { useState, useEffect } from "kaioken"

function MyComponent({ initialValue }) {
  const [value, setValue] = useState(initialValue)

  useEffect(() => {
    // This effect will run whenever 'value' changes
    console.log("Value has changed:", value)
  }, [value])

  const handleClick = () => {
    setValue(prevValue => prevValue + 1)
  };

  return (
    <div>
      <p>Value: {value}</p>
      <button onclick={handleClick}>Increment</button>
    </div>
  )
}`} />
</div>
