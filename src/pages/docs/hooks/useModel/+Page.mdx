# useModel

<CalloutBlock variant="warning">
 <span className="inline-block badge badge-warning mb-1">Deprecated</span>

  `useModel` has been deprecated and will be removed in a future release.

  It was introduced before [Two Way Binding](https://kaioken.dev/docs/api/signal#two-way-binding), which fits the intended purpose much better.
  For more complex form-related scenarios, consider using the [Form API](https://kaioken.dev/docs/api/form).

</CalloutBlock>

Similar to <a href="/docs/hooks/useRef">useRef</a>, but allows
you to easily set up two-way binding for a given input, textarea or select element.

<CodePreview data={useModelTypePreview} /> takes one argument - 
``initialValue``, and returns the following:
<ul>
  <li><b>elementRef</b>: A <CodePreview data={refTypePreview} /> used to bind to the target element.</li>
  <li><b>value</b>: The element's current value (alternatively gives the value of 'checked' if used on a checkbox / radio)</li>
  <li><b>setValue</b>: function provided to enable setting the value from code.</li>
</ul>
