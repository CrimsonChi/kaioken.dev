# useAsync

Similar to <a href="/docs/hooks/useMemo">useMemo</a>,
but runs an async function and returns <CodePreview data={UseAsyncStateTypePreview} />.

`useAsync` creates an <ExternalLink href="https://developer.mozilla.org/en-US/docs/Web/API/AbortController" external>AbortController</ExternalLink> 
and passes the controller's abort signal to the async function.
When `useAsync` re-runs, the signal is aborted, making it easy to cancel a running async function or request.
